# GrepTips

Personal blog built with [Hugo](https://gohugo.io/) and hosted on GitHub Pages.

## Requirements

- Hugo v0.157+

## Development

Start the local dev server with live reload:

```
hugo server
```

Include draft posts:

```
hugo server -D
```

The site is available at http://localhost:1313.

## Build

Generate the static site into the `public/` directory:

```
hugo
```
